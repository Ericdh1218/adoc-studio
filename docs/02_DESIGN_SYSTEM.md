# ADOC Studio | Design System

## 1. Identidad

ADOC Studio es una firma joven de arquitectura con una identidad elegante, minimalista y precisa.

La marca debe transmitir:

- Elegancia
- Confianza
- Precisión
- Profesionalismo
- Juventud
- Seguridad
- Potencial

La experiencia debe hacer que el visitante piense:

> “Estos saben exactamente lo que hacen.”
> “Quiero trabajar con ellos.”

---

## 2. Personalidad

ADOC Studio se percibe como un equipo joven, elegante, seguro y con ganas de demostrar su talento.

No busca impresionar con exceso, sino con precisión.

### Concepto principal

**Elegancia silenciosa.**

---

## 3. Principios visuales

- Minimalismo extremo.
- Mucho espacio en blanco.
- Pocos colores.
- Tipografía refinada.
- Animaciones sutiles.
- Fotografías limpias.
- Composición editorial.
- Ritmo visual lento y elegante.

---

---

## 5. Tipografía

### Display

Usada para títulos grandes, hero y frases principales.

- Metrophobic
- Peso ligero o regular
- Mucho espacio
- Mayúsculas solo cuando tenga intención editorial

### Body

Usada para textos largos y descripciones.

- Source Serif 4
- Elegante
- Lectura pausada
