# ADOC Studio | Project Vision

ADOC Studio es una plataforma web para una firma joven de arquitectura.

El objetivo no es crear únicamente una página informativa, sino una experiencia digital elegante, minimalista y profesional.

## Objetivo principal

Transmitir confianza, precisión y elegancia para convertir visitantes en clientes potenciales.

## Stack

- React
- Vite
- TypeScript
- pnpm
- Tailwind CSS
- Supabase
- React Router
- Framer Motion

## Filosofía

Construir un proyecto escalable, limpio y mantenible.

No copiar código sin entenderlo.

Cada tecnología, carpeta, componente y decisión visual debe tener un propósito claro.
